ABOUT CSS
  • CSS means Cascading Style Sheets.
  • CSS is used to style the HTML document.
  • In CSS,there are 3 ways of giving styles.
    1. External styles
    2. Internal styles
    3. Inline styles
  • External styles are styles which are in external style sheet and we have to extract them by providing the link in head section.
  • Internal styles are given within the HTML document and these are given in the head section.
  • Inline styles are specified for each HTML element in the document.
