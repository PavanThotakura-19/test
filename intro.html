About html

HTML is abbrevated as Hyper Text Markup Language. It is the standard markup language for creating Web pages. It describes the structure of a Web page. It consists of a series of elements. The elements tell the browser how to display the content. The element label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
